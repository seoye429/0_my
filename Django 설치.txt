Django 프로젝트 만들기

파이창에서 디장고 설치
터미널 설치:

1)가상환경 만들기
  cd C:\(슬러시 반대방향)       ->c드라이브로 이동
  python -m venv pythonPortfolio       -> port폴더 만들기

2)myworld를 열어준다  ->파이창에서 폴더 열기

3)django설치
  python -m pip install Django
  python -m pip install --upgrade pip   ->버전 업그레이드

4)프로젝트 생성-admin생성
  django-admin startproject my_portfolio  ->메인 my_portfolio은 프로젝트명,안의 my_portfolio는 admin

5)해당실행할 폴더로 이동
  cd my_portfolio

6) app을 만듬
 python manage.py startapp members   ->members   app이름

7)생성한 프로젝트 실행
 python manage.py runserver

8)python manage.py migrate  ->소스전달

from django.http import HttpResponse
from django.template import loader

def witch(request):
    template = loader.get_template('witch.html')
    return HttpResponse(template.render())

from . import views

    path('witch/', views.witch, name='witch'),
]
